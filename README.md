# VL.NAILS

Готовый статический сайт-визитка для GitHub Pages.

## Публикация
1. Загрузите содержимое папки `VL-NAILS` в репозиторий GitHub.
2. Откройте **Settings → Pages**.
3. Выберите публикацию из ветки `main`, папка `/root`.

## Быстрое редактирование
- Цены, тексты и контакты: `index.html`.
- Цвета и оформление: переменные в начале `style.css`.
- Фотографии: `images/gallery/`.
- Интерактивность: `script.js`.

Для корректного Open Graph после публикации рекомендуется заменить относительный путь `images/og-cover.webp` в `index.html` на полный адрес изображения на вашем домене.
